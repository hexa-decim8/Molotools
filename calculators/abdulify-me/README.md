# Abdulify Me

This project is maintained as a WordPress plugin under [wordpress](wordpress).

For installation and release guidance, see [wordpress/README.md](wordpress/README.md).

## Color Theming

The Abdulify Me widget is host-theme aware. When embedded on a page, it reads host CSS custom properties first and falls back to plugin defaults if those variables are not present.

See [wordpress/README.md](wordpress/README.md) for supported color variables and WordPress filter hooks.
