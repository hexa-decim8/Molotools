## [0.1.0] - 2026-04-28

### Added
- Initial release: Express API server with SQLite backend
- Campaign, content, and share toolkit management
- Social platform integrations: Facebook, Instagram, X (Twitter), TikTok
- OAuth 2.0 flows for connecting supporter accounts
- Share/click event tracking and aggregated analytics
- Embeddable vanilla JS share widget (`/embed/toolkit.js`)
- Database migrations with schema versioning
- Seed script for demo data
- Health check endpoint
